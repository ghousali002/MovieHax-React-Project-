Welcome to **MovieHax**, your one-stop destination for exploring, discovering, and enjoying movies of all genres. This web application is designed to enhance your movie-watching experience by providing a user-friendly interface to search for, explore, and learn more about your favorite films. Whether you're a film buff or just looking for something to watch tonight, we've got you covered.


## Features

- **Trending Movies**: Stay up-to-date with the latest trending movies that everyone's talking about.

- **Genre Exploration**: Dive into specific genres to discover hidden cinematic gems tailored to your preferences.

- **Search Functionality**: Quickly find movies by title, making it effortless to locate your favorites.

- **Detailed Movie Information**: Access comprehensive details for each movie, including IMDb ratings, release dates, and captivating descriptions.

## Technologies Used

- **React**: The project is built using React, a popular JavaScript library for building user interfaces.

- **The Movie Database (TMDb) API**: We leverage TMDb API to fetch and display movie data, ensuring access to an extensive collection of films.

- **Bootstrap**: Bootstrap CSS framework is used for responsive and attractive styling.

## Getting Started

Follow these steps to get "MovieHax" up and running on your local machine:

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/ghousali002/moviehax.git
   ```

2. Navigate to the project directory:

   ```bash
   cd moviehax
   ```

3. Install project dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm start
   ```

5. Open your web browser and visit [http://localhost:3000](http://localhost:3000) to explore "MovieHax."
